## C语言中可变参数函数实现原理

### C函数调用的栈结构
  可变参数函数的实现与函数调用的栈结构密切相关，正常情况下C的函数参数入栈规则为__stdcall,它是从右到左的，即函数中的最右边的参数最先入栈。
```
void fun(int a, int b, int c)
{
  int d;
  ...
}
```

  其栈结构如下:
```
  0x1ffc-->d
  0x2000-->a
  0x2004-->b
  0x2008-->c
```
