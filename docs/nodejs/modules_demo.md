NodeJS常用模块介绍
===========

一. restify
----------------------
  Restify是nodejs构建REST web服务的模块。它从express借鉴实现，主要满足RESTful API构建。
  
二. connect
----------------------
  Connect是node的一个可扩展HTTP服务器， 使用所谓中间件的插件系统。
